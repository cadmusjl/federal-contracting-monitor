================================================================================
                         API ENDPOINTS
================================================================================

Once deployed, the following endpoints will be available:

1. GET /api/opportunities
   - Query: ?page=1&limit=25&agency=DoD&set_aside=8a&source=db
   - Response: List of federal opportunities
   - Source: Database (db) or SAM.gov Live (sam)

2. GET /api/awards
   - Query: ?page=1&limit=25&agencies=DoD&source=db
   - Response: List of contract awards and spending
   - Source: Database (db) or USAspending.gov Live (usaspending)

3. POST /api/sync
   - Body: {"action":"sync-opportunities","source":"sam.gov"}
   - Response: Sync status and record count
   - Purpose: Trigger manual data synchronization

4. GET /api/filters
   - Query: ?user_id=xxx
   - Response: User's saved filter configurations
   - Purpose: Retrieve saved search filters
