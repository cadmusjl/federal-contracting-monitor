TROUBLESHOOTING GUIDE
================================================================================

Issue: "Repository not found" in Vercel
Solution: Make sure GitHub account is authenticated to Vercel, repo is public

Issue: "Environment variables not loading"
Solution: Verify NEXT_PUBLIC_ prefix on variables (required for client-side)

Issue: "Supabase connection refused"
Solution: Check NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY values

Issue: "Domain not resolving"
Solution: Wait 24-48 hours for DNS, check with: nslookup federal-monitor.innovixdynamix.com

Issue: "Build fails"
Solution: Check Vercel build logs for specific error, likely missing env var
